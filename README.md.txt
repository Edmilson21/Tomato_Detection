# Detección de Tomates con YOLOv8

## Descripción

Este proyecto utiliza YOLOv8 para detectar tomates en imágenes. El modelo fue entrenado y validado utilizando un conjunto de datos etiquetado manualmente.

## Conjunto de Datos

Fuente: [Open Images Dataset](https://storage.googleapis.com/openimages/web/index.html)
Número de Imágenes de Entrenamiento: 300
Número de Imágenes de Validación: 30

## Herramientas Utilizadas

1- YOLOv8: Para la detección de objetos.
2- LabelImg: Para etiquetar las imágenes.
3- OpenCV: Para el procesamiento de imágenes y video.
4- Google Colab: Para entrenamiento en GPU.

## Resultados

El modelo ha logrado una precisión del 90% en la detección de tomates en el conjunto de validación.
